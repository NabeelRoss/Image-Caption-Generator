# AI Image Caption Generator

A smart web application that generates captions for images using the **BLIP** model and rewrites them in different styles (Instagram, Poetic, Professional) using **Flan-T5**.

## Features
- **Visual Recognition:** Uses Salesforce's BLIP model to understand image content.
- **Style Transfer:** Uses Google's Flan-T5 to convert factual descriptions into creative captions.
- **Styles:** Social Media, Professional, Poetic, and Roast.
